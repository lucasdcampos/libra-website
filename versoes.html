2025

Libra 1.0.0-Beta

Primeira versão disponível para ser baixada oficialmente. Contém o básico que uma linguagem precisa: variáveis, condicionais, loops, funções, vetores e uma API simples para se comunicar com o C# nativo.

Essa versão possui diversos problemas e sofre de muita falta de otimização, mas é totalmente usável e faz o que se propõe a fazer: ensinar programação à iniciantes e falantes da lingua portuguesa.

2024

Primeiro Protótipo - Jun

O primeiro protótipo da Libra foi criado em poucas semanas, ele foi um experimento para uma apresentação de faculdade. A sintaxe inicial era uma mistura de Lua com C e não havia nenhum plano a ser seguido.

Não havia intenção de crescer o projeto até ele virar o que é hoje.
